# ☕ Claude 續跑模式（keep-awake tray）

一個 Windows 系統匣小工具：**闔上筆電時讓工作（例如 Claude Code）繼續跑，用完自動還原並讓筆電睡覺。**
不需系統管理員權限、本體不連網、不收集任何資料，只用內建的 `powercfg` 改「闔蓋動作」與「電池閒置睡眠」。

---

## 安裝

### 方法 A：一行指令（給熟悉終端機的人）
開 **PowerShell**，貼上執行：
```powershell
irm https://raw.githubusercontent.com/chuanchihsu1219/keep-awake/main/install.ps1 | iex
```

### 方法 B：雙擊安裝（給一般使用者）
1. 按綠色 **Code → Download ZIP**，下載後解壓縮。
2. 進資料夾，**雙擊 `install.bat`**。
3. 完成後右下角系統匣會出現咖啡杯 ☕ 圖示，並設定開機自動啟動。

> 安裝位置：`%LOCALAPPDATA%\ClaudeKeepAwake`

### 解除安裝
雙擊 **`uninstall.bat`**（會停止程式、還原電源設定、移除開機捷徑與檔案）。

---

## 怎麼用
- 系統匣咖啡杯 ☕：**左鍵**切換開/關、**右鍵**選單。
- 時長：`30 分鐘`（左鍵預設）、`1 小時`、`3 小時`、`直到電力耗盡`。
- **倒數從「闔上筆電」才開始**；按下後 5 分鐘沒闔上會自動取消並還原。
- 三道自動還原：**掀開筆電** → 立即還原（通知續跑了多久）；**計時到期** → 還原，若你人不在則讓筆電睡眠；**開機時** → 若上次沒正常結束則自動還原。
- 圖示：綠色冒煙＝續跑中、灰色＝關閉。

## 🔒 這是什麼、安不安全？
這是**純文字的 PowerShell 腳本**，沒有編譯過的執行檔、沒有黑盒子——你（或你們 IT）可以直接打開每個檔案看它做什麼。

**它會做的**（只有這些）：
- 用 `powercfg` 把「闔蓋動作」暫時設為「不做任何動作」、電池閒置睡眠暫時設為「永不」。
- 關閉 / 掀開 / 到期時，把上述兩項**改回正常**。
- 到期且你人不在時，呼叫系統的睡眠。

**它不會做的**：
- ❌ 不需要系統管理員權限
- ❌ 本體不連任何網路（只有「一行安裝」那步會從這個 GitHub 下載檔案）
- ❌ 不讀你的檔案、不收集或上傳任何資料
- ❌ 不常駐背景服務，就一個系統匣小程式，隨時可從選單「結束」

**防毒 / SmartScreen 提醒**：因為它用 `.vbs` 隱藏啟動 PowerShell（為了開機不閃黑窗），少數防毒的啟發式偵測可能會提示。這是腳本類工具的常見情況，你可以：
- 直接讀原始碼確認（就在這個 repo）
- 掃描報告：**VirusTotal → _<掃描後把連結貼這裡>_**
